Usage in this app:
  Bundled sample footage used ONLY by the optional live-analysis "demo feed",
  which is gated behind the presence of a `demo_feed` flag file in the app's
  external files dir. In normal/production use the flag is absent and the live
  device camera is used instead. The demo feed exists so the real Gemini
  form-analysis pipeline can be exercised on emulators / CI where no physical
  camera or exerciser is available.
